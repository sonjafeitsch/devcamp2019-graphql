FENNEC

What is the FENNEC?

  • Web-based workbench for adding traits to community tables.
  • Basic tools for interactive visualization.
  • Public instance at http://fennec.molecular.eco
  • Integrated database with 1.6 million organisms and 207000 trait entries
  • Run it locally using docker container.
